# SpotMyLyrics
A simple Java application to fetch lyrics of the current Spotify songs (MacOS only)

## Getting Started

Clone the repo, run
```chmod +x SpotMyLyrics```
then
```./SpotMyLyrics```

### Prerequisites

You'll need Java 8>.

### Installing

No specific installation is required.
